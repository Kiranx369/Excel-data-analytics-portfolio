# Comprehensive Excel Data Analytics & Data Engineering Portfolio

Welcome to my professional data analytics portfolio. This master repository showcases a structured, progressive curriculum tracking my development from fundamental data sanitation to advanced relational database architecture and live API pipeline deployment.

---

## 🧭 Portfolio Navigation & Project Framework

### 📂 [Project 1: Telecom Customer Churn Cleaning](./project-1-telecom-churn-cleaning/)
*   **Core Mechanics:** Row deduplication, string trimming, data type alignment, and missing value reconciliation.
*   **Analytical Objective:** Establishing raw baseline data integrity for downstream reporting layers.

### 📂 [Project 2: HR Corporate Employee Attrition Dashboard](./project-2-hr-employee-attrition-dashboard/)
*   **Core Mechanics:** Single-table Pivot Table calculations, interactive multi-chart slicer panels, and workforce behavioral tracking.
*   **Analytical Objective:** Isolating organizational risk factors and retention imbalances across corporate departments.

### 📂 [Project 3: Multi-Table Supply Chain Logistics Auditor](./project-3-supply-chain-logistics-auditor/)
*   **Core Mechanics:** Relational database design (Star Schema), Power Pivot Data Modeling, and Many-to-One bridge tables.
*   **Analytical Objective:** Dissecting global freight distribution networks and cross-examining fulfillment load vs. physical warehouse asset capacities.

### 📂 [Project 4: City of Chicago Municipal Salary & Budget Audit](./project-4-chicago-salary-budget-audit/)
*   **Core Mechanics:** Live API server streaming endpoints, case-sensitive text processing (Power Query M-code), and labor scale normalization.
*   **Analytical Objective:** Auditing thousands of public public record datasets to evaluate true taxpayer expenditure distribution across municipal operations.
